# Pima Diabetes EDA Dashboard

Pima Indians Diabetes 데이터의 숨은 결측, 진단 그룹별 분포, 변수 관계와 로지스틱 회귀 임계값을 탐색하는 Streamlit 대시보드입니다.

> 교육용 분석이며 실제 의료 진단이나 치료 결정에 사용할 수 없습니다.
